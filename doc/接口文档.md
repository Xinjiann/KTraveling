## 全局统一返回格式
#### 无返回值情况
{\
&emsp;    "msg": "success",\
&emsp;    "code": 0\
}

#### 有返回值情况
{\
&emsp;    "msg": "success",\
 &emsp;   "code": 0,\
 &emsp;   "data": [\
 &emsp;&emsp;       "London Luton Airport Airport Way, Luton LU2 9LY, United Kingdom"\
  &emsp;  ]\
}

错误码统一返回code 500。错误结果无data字段，错误信息放在msg里

}

## 8. 登陆

#### 1. 接口描述    

前端调用wx.login()生成code，携带code访问接口

#### 2. 请求地址  

`/ktravel/auth/login`  

#### 3. 请求方式  

**POST**  

#### 4. 请求参数  
body参数

| 参数名       | 必选 | 类型      | 示例         |
| ------------ | ---- | ---------------- | ------------ |
| code|    是   |   string     | '123' |

#### 5. 返回示例 
{\
&emsp;    "msg": "success",\
 &emsp;   "code": 0,\
 &emsp;   "data": 123456
}

此处data是返回的token, 后续访问需要讲token携带在header里，后端根据redis储存的token信息进行拦截判断
